Project History (1)

The Real Okotoks Pizza Project: Developed as a mandatory assignment at InnoTech College in August 2020. This project required me to plan, design, create and deploy a complete website to the Internet.

I used my solid knowledge of CSS and HTML to create and style the contents of each web component. Wrote distinctive descriptions for each menu item to target potential customers for the products. Project History (2)

The Full-Stack Tweeter Project: Developed as a mandatory assignment at InnoTech College in November 2020. This project required me to plan, design, create and deploy a fully functional mockup of Twitter to the Internet.

For the front-end, I used VueX, VueJs to write all the code that would be handling all Axios Requests (i.e.: fetching, posting, patching, deleting) the data directly from/to the API backend provided by the instructor.

For the back-end, I used MariaDB, DBeaver, SQL, Python, Postman, and re-wrote some of the front-end code. For the complete deployment to Google Cloud/Internet, I used my Apache2 server, Flask, PM2, Certbot, and GitHub. Acquired more front-end and back-end web development skills such as creating a dedicated database and server.

Here's the link to the live website: https://thefullstacktweeterproject.ml/#/
