In order to earn my diploma as a full-stack web developer at InnoTech College, I was required to plan, design, create and deploy three (3) major projects.

The Real Okotoks Pizza Project: Developed as an intro assignment. I used my solid knowledge of CSS and HTML to create and style the contents of each web component. Wrote distinctive descriptions for each menu item to target potential customers for the products. Conducted a thorough research and selection of images to enrich the content of each web page. Acquired front-end and back-end web development skills such as creating a dedicated server.

The Full-Stack Tweeter Project:Developed as a fully functional mockup of Twitter. For the front-end, I used VueJs to write all the code that would be handling all Axios Requests (i.e.: fetching, posting, patching, deleting) the data directly from/to the API backend provided by the instructor. For the back-end, I used MariaDB, DBeaver, SQL, Python, Postman, and re-wrote some of the front-end code. For the complete deployment to Google Cloud/Internet, I used my Apache2 server, Flask, PM2, Certbot, and GitHub. Acquired more front-end and back-end web development skills such as creating a dedicated database and server.

The Full-Stack MVP Project: Developed as a final project to showcase all skills and tools learned throughout this program. For the front-end, I used VueJs to write all the code that would be handling all Axios Requests (i.e.: posting) the data directly from/to the API backend. For the back-end, I used MariaDB, DBeaver, SQL, Python, Postman, and Google Chrome tools. For the complete deployment to Google Cloud/Internet, I used my Apache2 server, Flask, PM2, Certbot, and GitHub. Acquired more front-end and back-end web development skills such as creating a dedicated database and server.
